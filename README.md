# منصة الألعاب المتكاملة بـ 5 ملفات فقط 🎮

منصة ألعاب احترافية مبنية باستخدام **Node.js + Express + SQLite** وجاهزة للرفع المباشر على **Render.com**.

## المميزات
- **لوحة تحكم كاملة `/admin`**: تعديل اسم الموقع، الألوان، الأزرار، التليجرام، وإدارة المحاولات مباشرة.
- **نظام محاولات لعب ذكي**: 6 محاولات يومية لكل مستخدم، وتتصفّر تلقائياً الساعة 12 منتصف الليل بتوقيت اليمن (`Asia/Aden`).
- **3 ألعاب افتراضية مدمجة**: (الثعبان، المدافع، 2048) تنشأ تلقائياً عند أول تشغيل.
- **دعم ميزة Wake Lock API**: لمنع انطفاء شاشة الهاتف أثناء اللعب.
- **رابط `/ping`**: جاهز للربط مع خدمات مثل `UptimeRobot` لضمان بقاء السيرفر نشطاً.

## طريقة الرفع على Render.com
1. قم بإنشاء مستودع (Repository) جديد على GitHub وارفع الملفات الخمسة إليه.
2. في موقع Render، قم بإنشاء **Web Service** جديدة واربطها بالمستودع.
3. **أمر البدء (Start Command):** `npm start`
4. **المتغيرات البيئية (Environment Variables):** أضف في قسم الـ Environment:
   - `ADMIN_PASSWORD`: كلمة مرور لوحة التحكم.
5. **مهم جداً للاستمرارية (Disk Persistence):**
   - اذهب إلى إعدادات الخدمة في ريندر وأضف **Disk** بمساحة 1GB (أو أي مساحة صغيرة).
   - اجعل مسار الربط (Mount Path) هو `/data`. هذا يضمن عدم حذف قاعدة البيانات والألعاب المرفوعة عند كل Deploy جديد.
